# vantumiqp Landing Page Design

## Goal

Create a single-page marketing site for vantumiqp that borrows the refined, editorial feel of Bluefish AI while clearly presenting a business-intelligence product and funneling visitors toward demo access.

## Product framing

- vantumiqp is a rebranded Apache Superset experience.
- The current offer is demo access only.
- There is no pricing section, checkout path, or multi-page marketing funnel yet.

## Chosen direction

Use a **cinematic editorial analytics** direction:

- large painterly image surfaces from the supplied assets
- crisp monochrome structure with oceanic cyan accents
- generous whitespace and asymmetrical card layouts
- minimal navigation
- a final dark CTA region with the demo-access form

This keeps the visual kinship with Bluefish AI without copying its messaging or irrelevant marketing/AI diagrams.

## Page structure

1. **Top navigation**
   - wordmark
   - anchors for Product, Workflow, Demo
   - primary CTA for demo access

2. **Hero**
   - painterly background image
   - eyebrow noting Apache Superset foundation
   - concise headline and supporting copy
   - primary and secondary CTAs
   - compact proof points

3. **Product capabilities**
   - four shadcn cards for dashboards, SQL exploration, charting, and governed sharing
   - supporting imagery using the supplied painterly assets

4. **Workflow section**
   - a clean flow from query to charts to dashboards to decisions
   - faux product UI panels to suggest the Superset experience without inventing real screenshots

5. **Demo access form**
   - fields for name, work email, company, and use case
   - no pricing language
   - clear expectation-setting copy

6. **Footer**
   - minimal legal/product links
   - brand close

## Visual system

- **Typography:** Geist body copy with a softer editorial display face for larger headings.
- **Palette:** neutral paper backgrounds, near-black text, cool cyan accents, and dark marine overlays.
- **Motion:** restrained entrance animation, soft hover lifts, and subtle image scaling only.
- **Components:** use shadcn primitives for buttons, cards, badges, separators, and form controls.

## Content notes

- Mention Apache Superset honestly as the foundation.
- Emphasize dashboards, SQL exploration, charting, and governed reporting.
- Avoid pricing, generic startup fluff, and Bluefish-specific AI language.

## Success criteria

- Feels visually adjacent to Bluefish AI at a glance.
- Reads unmistakably as a BI product page.
- Has one obvious conversion path: request demo access.
- Uses supplied assets tastefully without embedding Bluefish-branded imagery.
